# setAppFull（设置应用全屏）
set app to full screen run.<br>
将应用突破次元壁，覆盖异形屏幕限制全面显示_(:з」∠)_<br>
强制让异性屏幕的手机运行游戏时可以全屏显示无黑边<br>
勾选应用重启系统即可<br>
